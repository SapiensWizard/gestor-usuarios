# CRUD - Gestor de Usuários

Sistema Web para gestão de usuários desenvolvido em **PHP**, utilizando **Programação Orientada a Objetos (POO)** e banco de dados **MySQL**. O projeto implementa as operações básicas de um CRUD (Create, Read, Update e Delete), além de pesquisa, autenticação e uma interface moderna com Bootstrap.

## Funcionalidades

- Cadastro de usuários
- Listagem de usuários
- Pesquisa de usuários
- Visualização de detalhes
- Edição de usuários
- Exclusão de usuários

## 🚀 Tecnologias Utilizadas

### Back-end

- PHP 8+
- Programação Orientada a Objetos (POO)
- MySQL
- MySQLi(MySQL Improved)

### Front-end

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- JavaScript (JS)
- jQuery
